Pneumonia Detector

Welcome to the pneumonia detector.

This is a simple web app that uses a machine learning model to detect pneumonia in chest x-rays.

To get started, click the button below, you will redirect to the detector page. There, click the button to upload an image of a chest x-ray and click the "Detect pneumonia" button to run the model.

The model will then tell you whether the x-ray shows signs of pneumonia or not.

The model was trained using the Chest X-Ray Images (Pneumonia) dataset from Kaggle.
